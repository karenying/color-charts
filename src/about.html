Color Charts

v1.2.1

Visuals that depend on colors to convey information may be hard to read for people with color blindness. This extension aims to improve upon existing extensions by using researched color blind friendly palettes. It allows for filtering of visuals on aweb page, with the option to choose between 4 different color blind friendly palettes.

Built with vanilla JavaScript, Color Charts interacts with HTML <img> elements in the Chrome browser, utilizing HTML canvas and base64 encoding to filter the images.
